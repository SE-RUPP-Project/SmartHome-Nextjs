## 1. Install Dependencies (2 min)

```bash
npm install
```

## 2. Add shadcn/ui Components (3 min)

Add all core UI components in one command:

```bash
npx shadcn@latest add button card input label select dialog dropdown-menu avatar badge separator switch toast tabs alert-dialog
```

Or add them individually:
```bash
npx shadcn@latest add button
npx shadcn@latest add card
npx shadcn@latest add input
# etc...
```

## 3. Update Login Page (1 min)

Open `src/app/login/page.tsx` and uncomment the shadcn/ui code (lines 19-66).

Comment out the placeholder code (lines 70-123).

## 4. Run! (Instant)

```bash
npm run dev
```

Open http://localhost:3000
